**aiozipkin** is Python 3.5+ module that adds http://zipkin.io/ insturementation
from asyncio applications


Installation
------------
Installation process is simple, just::

    $ pip install aiozipkin



Requirements
------------

* Python_ 3.5+
* aiohttp_
